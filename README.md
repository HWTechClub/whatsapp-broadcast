# whatsapp-broadcast
Whatsapp bot that sends tech club updates to a individual user  


## Install go compiler 
https://golang.org/doc/install

## Install libraries
```
go install github.com/Baozisoftware/qrcode-terminal-go
go install github.com/Rhymen/go-whatsapp
```

## Run 
```
go run broadcast.go
```
